**Security & Anonymization**
User privacy and data security are the top priorities of this project.
- **Encrypted IDs:** The Player/Account IDs stored in the public history logs are **encrypted/hashed**. This ensures that raw account identifiers are not directly exposed in the repository, maintaining user privacy while allowing for statistical tracking.
- **Secure Tokens:** Sensitive authentication data (Tokens, Client Secrets) are stored securely within **GitHub Secrets** and are never exposed in the source code or output files.

**Open Data for Developers**
This repository automatically generates daily leaderboard data in **JSON format**.
We encourage other developers, data analysts, and gamers to use this data! You are free to:
- Pull the JSON files from this repo.
- Build your own charts, graphs, or leaderboard visualizers.
- Analyze the scoring trends of Fortnite Festival.
